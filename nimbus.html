Overview

Nimbus is a small, self-hosted cloud platform for one: an Elixir and Phoenix app that manages the Proxmox + k3s cluster living in my house. It's opinionated on purpose — every VM it creates gets the same minimal NixOS image (small, medium, or large, one CPU unit each), installed unattended over SSH with nixos-anywhere, so a new box comes out identical to the last one instead of being its own hand-tuned snowflake.

Provisioning VMs turned out to be the easy part. Nimbus also runs its own CI: it builds Docker images for its own deployments (and everything else I write) using Kaniko inside the cluster, pushes them to a self-hosted registry, and redeploys the k3s workloads — so pushing a change to Nimbus is how Nimbus updates itself. Beyond that it manages Postgres clusters (CloudNativePG), persistent storage (Longhorn), monitoring (Prometheus and Grafana), and PXE boot images for bringing new hardware online.

None of this is because I distrust a hosted alternative — it's because I wanted a cluster made of exactly the pieces I actually use, running on hardware I already own, with no bill at the end of the month.
